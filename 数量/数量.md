# 多元线性回归

## 假设
1. 线性关系
2. 残差同方差（违反就是异方差）
3. 残差独立（违反就会出现序列相关）
4. 残差服从正态分布
5. 自变量之间不相关（违反就是多重共线性）
6. 自变量和残差之间不相关
## 违反假设检验
1. 因变量与自变量的散点图：可以检验是否存在非线性关系（正常应该是一条职衔）
2. 残差散点图 ：检验和残差相关的性质。（如果随着x增大而扩散，则违反同方差；如果残差出现周期性，则出现序列相关；如果残差与某个自变量有明显相关趋势，则违反第6条）
3. 散点矩阵：识别极端值，异常值
4. Q-Q图：判断正态分布
## 公式

$$
Y = b_0 + b_1X_1 + b_2X_2 + \cdots + b_kX_k + \varepsilon
$$

- $k$ = number of **independent** variables ($k=1$ simple linear regression)
- $n$ = number of observations ($n$ must $> k$)
- $b_0$ = intercept: estimated value of $Y$ when $X_1, X_2,\dots, X_k$ are all equal to zero.
- $b_1,\dots, b_k$ = partial slope coefficients: the expected change in the dependent variable for a 1-unit increase in an independent variable, **holding all the other independent variables constant**.
- $\varepsilon$=error/residual term: the stochastic or random part of the model.

估计项不需要加残差项了，变量上加上hat。

$$
\hat{Y} = \hat{b}_0 + \hat{b}_1\hat{X}_1 + \hat{b}_2\hat{X}_2 + \cdots + \hat{b}_k\hat{X}_k
$$

## 拟合优度度量（判断它是否具备基本的解释价值，且需要和基准进行比较）
### **ANOVA table**：分解总变异（SST）为回归变异（SSR）和残差变异（SSE），从方差分解角度看模型解释力。
![alt text](image.png)
![alt text](image-1.png)

### $R^2$

$$
R^2 = \frac{RSS}{SST} = 1 - \frac{SSE}{SST}= \frac{\text{explained variation}}{\text{total variation}} = 1 - \frac{\text{unexplained variation}}{\text{total variation}}
$$

- ✓ $0 \le R^2 \le 1$
- ✓ The higher $R^2$, the better fitness.

### Adjusted R² ($\bar{R}^2$)

$$
\bar{R}^2 = 1 - \frac{SSE/(n-k-1)}{SST/(n-1)} = 1 - \left[ \left( \frac{n-1}{n-k-1} \right) \times (1 - R^2) \right]
$$

 R²>Adjusted R²;调整R²可能小于0，两者都无法提供系数是否统计显著的信息，可以引入F分布；无法提供估计系数 / 预测是否有偏的信息；无法判断模型拟合是否 “好”（R² 的高低没有绝对标准）

### AIC&BIC
- **AIC**:

$$
\text{AIC} = n \ln\left( \frac{\text{Sum of squares error}}{n} \right) + 2(k + 1)
$$

- **BIC**:

$$
\text{BIC} = n \ln\left( \frac{\text{Sum of squares error}}{n} \right) + \ln(n)(k + 1)
$$

AIC更关注解释力度
BIC更关注简洁程度，惩罚项更强

## 回归系数显著性检验（在拟合优度不错的基础上，排除 “偶然拟合” 的可能）
### 对单个因子进行检验
- **原假设（Null Hypothesis）**：

$$
H_0: b_1 = \text{hypothesized value of } b_1
$$

- **检验统计量（Test Statistic）**：

$$
t = \frac{\hat{b}_1 - \text{hypothesized value of } b_1}{S_{\hat{b}_1}}, \quad \text{df} = n - k - 1
$$

- $\hat{b}_1$：自变量$X_1$对应的回归系数估计值
- $S_{\hat{b}_1}$：回归系数$\hat{b}_1$的标准误（standard error）
- $\text{df}$：自由度（degrees of freedom）
- $n$：观测值数量
- $k$：模型中自变量的总个数
###  Joint F-test
- **原假设（Null Hypothesis）**：

$$
H_0: b_m = b_{m+1} = \dots = b_{m+q-1} = 0
$$

- **备择假设（Alternative Hypothesis）**：

$$
H_a: \text{At least one slope of the } q \text{ slopes} \neq 0.
$$

- **符号说明**：
  - $m$：第一个受约束的斜率（the first restricted slope）
  - $m+1$：第二个受约束的斜率，依此类推，直到第 $q$ 个受约束的斜率

$$
\text{F-statistic} = \frac{(\text{Sum of squares error restricted model} - \text{Sum of squares error unrestricted})/q}{\text{Sum of squares error unrestricted model}/(n - k - 1)}
$$


- $q$：受约束的斜率系数个数
- $n$：观测值数量
- $k$：无约束模型中自变量的总个数
- **Sum of squares error restricted model**：约束模型的残差平方和（SSE restricted）
- **Sum of squares error unrestricted model**：无约束模型的残差平方和（SSE unrestricted）

###  General linear F-test
- **原假设（Null Hypothesis）**：

$$
H_0: b_1 = b_2 = b_3 = \dots = b_k = 0
$$

- **备择假设（Alternative Hypothesis）**：

$$
H_a: \text{at least one } b_j \neq 0 \quad (\text{for } j = 1, 2, \dots, k)
$$

$$
F = \frac{MSR}{MSE} = \frac{RSS/k}{SSE/(n - k - 1)}
$$

- $MSR$：回归均方（Mean Square Regression）
- $MSE$：均方误差（Mean Square Error）
- $RSS$：回归平方和（Regression Sum of Squares，解释的变异）
- $SSE$：残差平方和（Error Sum of Squares，未解释的变异）
- $k$：模型中自变量的总个数
- $n$：观测值数量

## 错误
### 建立模型时的错误（会违反一致性，虽然增加样本容量估计不会更准）
![alt text](image-2.png)、
1. 遗漏变量：会导致残差中包含着自变量，导致方差上升，导致异方差；因为残差中可能包含着序列相关的自变量，就会导致残差也会序列相关
2. 不合适的变量形式：例如非线性关系使用线性回归，明显会导致方差上升，导致异方差
3. X不合适：例如某个X非常大，可能导致异方差
4. 不同时间的数据放在一起

## overfitting
1. 一般在监督学习下发生较多，generalize（泛化较差）
2. 三类错误：
Bias error：样本内数据都没训练好，钱你和
variance error：泛化差，过拟合
base error：误差项生成的，无法避免
3. 修正：加入惩罚项，cross-validation（多次验证）
## 监督学习（概念，适用条件）
### 连续——回归
1. 线性：LASSO（加惩罚项），传统OLS是让SSE最小，LASSO是让SEE+惩罚项最小
2. 非线性：regularization：正则化，降低x个数

### 离散——分类
#### SVM
通过确定离所有观测点最远的分类边界，最大化正确预测的概率，提升模型对新数据的泛化能力。
适用条件：线性数据；**二分法**；小的高维数据
![alt text](image-13.png)
![alt text](image-14.png)
#### K-nearest neighbor (KNN)
本质：找和他最近的k（超参数）个点，少数服从多数
缺点：
- K 太小：容易受噪声影响，过拟合；
- K 太大：会把远处无关样本算进来，欠拟合；
适用条件：线性数据，简单数据
#### CART
既可以分类又可以回归

#### Ensemble learning
- heterogeneous learners：不同算法，同一数据
- homogenous learners,：同一算法，使用bootstrap取出的不同数据，例如**Random forest**
